# Verilog2Tetris
Verilog2Tetris is based off the same architecture from the Nand2Tetris curriculum but instead of using a custom HDL everything is written in Verilog. The project is made for IceMaker and the default pin assignments are for the FOMU PVT development board.

## Current Functionality
Values must be manually set in top.v
