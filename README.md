# Verilog2Tetris
Verilog2Tetris is based off the same designs from the Nand2Tetris curriculum but instead of using a custom HDL everything is written in Verilog. The file layout is made for Xilinx Vivado and the default pin assignments are for the Arty 27-25 (Xilinx Spartan) development board.
### IMPORTANT: This branch will lag behind the main branch designed for the RZ-EasyFPGA A2.2. If you are planning on using this as a Vivado project please be aware it is not up to date.
## Current Functionality
Currently, the code runs the ALU from Nand2Tetris. There is a built in demo for the Arty S7-25.
